## Álgebra Lineal

En esta sección se trabaja el álgebra lineal con Python, partiendo desde una revisión de los tipos de datos básicos como listas, enteros y cadenas de texto, hasta operaciones avanzadas con vectores y cálculo simbólico.

### Vectores con NumPy y Matplotlib

Se desarrollan los conceptos fundamentales de vectores mediante NumPy para el cálculo numérico y Matplotlib para la representación gráfica. Los temas que se cubren incluyen:

- Vectores en 2D y 3D
- Norma de un vector
- Suma de vectores y multiplicación por un escalar
- Vector cero y vector unitario
- Normalización de vectores
- Producto punto
- Cálculo del ángulo entre dos vectores
- Proyección de un vector sobre un eje

Cada operación se acompaña de visualizaciones que permiten comprender el comportamiento geométrico de los vectores en el espacio, facilitando la conexión entre la formulación matemática y su interpretación visual.

### Cálculo simbólico con SymPy

Se introduce SymPy como herramienta para trabajar con expresiones algebraicas de forma exacta, sin aproximaciones numéricas. Esto permite resolver ecuaciones, calcular derivadas e integrales, y manipular expresiones matemáticas de manera simbólica, lo cual es especialmente útil cuando se requiere precisión analítica.

### Sistemas de ecuaciones y subespacios fundamentales

Se aborda la resolución de sistemas de ecuaciones lineales y se revisan los subespacios fundamentales asociados a una matriz, proporcionando la base teórica necesaria para temas más avanzados como descomposición en valores singulares, transformaciones lineales y reducción de dimensionalidad.
